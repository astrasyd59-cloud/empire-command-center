## 🛠️ FABRIC INTEGRATION PLAN

### What is Fabric?
- CLI tool by Daniel Miessler
- Pattern-based AI workflows
- Local-first (works with Ollama)
- Unix philosophy: pipe text through patterns

### How to Use with OpenClaw

**Option 1: Steal Patterns Only**
```bash
# Clone for pattern library
git clone https://github.com/danielmiessler/fabric
# Extract patterns for your agents
cp fabric/patterns/summarize/system.md ~/.openclaw/patterns/
```

**Option 2: Full Integration**
- Install Fabric locally
- Add patterns as agent skills
- Route tasks through Fabric + your agents
