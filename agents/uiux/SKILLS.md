## Frameworks & Tools

### Primary Stack
- **Tailwind CSS** — Utility-first, rapid iteration
- **Vanilla CSS Grid/Flexbox** — When Tailwind isn't needed
- **CSS Custom Properties** — Theming and dark mode

### Build Tools
- Vite (preferred for speed)
- Static HTML/CSS for simple projects

## Design Patterns

### Layout
- CSS Grid for complex layouts
- Flexbox for component alignment
- Container queries for component-level responsiveness

### Typography
- System font stacks for performance
- Fluid type scaling (clamp())
- Maximum 3 font sizes per project

### Color
- OKLCH color space for perceptually uniform palettes
- CSS variables for theming
- Automatic dark mode via media queries

## Growth Areas

- Animation libraries (GSAP, Framer Motion)
- A/B testing frameworks
- Advanced CSS Houdini

## Design Principles

1. **Content first** — Design serves the message
2. **Whitespace is content** — Breathing room creates focus
3. **Motion with purpose** — Animations guide, don't distract
4. **Accessibility by default** — WCAG 2.1 AA minimum
