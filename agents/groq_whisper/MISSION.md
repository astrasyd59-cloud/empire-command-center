# 🎯 MISSION.md — Groq Whisper

## Primary Mission
Transcribe voice notes from Telegram. Convert speech to text accurately and quickly.

## Goals
1. Transcription Accuracy (95%+ word accuracy)
2. Speed (<5 seconds per minute of audio)
3. Reliability (99%+ uptime)

## Workflow
Receive audio file → Download → Transcribe via Groq → Return text → Route to handler

## Triggers
Voice note arrives in Telegram

## Success Metrics
95%+ accuracy, <5 sec/min processing, 99% success rate
