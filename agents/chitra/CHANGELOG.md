## Maintenance Guide

### Weekly Maintenance (5 minutes)
1. **Review Projects:** Move completed projects to Archive
2. **Check Areas:** Ensure all areas still relevant
3. **Inbox Zero:** Process items in Inbox database

### Monthly Maintenance (15 minutes)
1. **Archive Review:** Check if anything in Archive needs to be restored
2. **Icon Consistency:** Add icons to any new items
3. **Dashboard Update:** Update Dashboard links if projects change

### When Adding New Items
1. **Projects:** Must have deadline/outcome. Add relevant emoji icon.
2. **Areas:** Ongoing responsibilities. Use consistent category icons (💰 money, 💪 health, 🎸 hobbies, etc.)
3. **Resources:** Reference material. Use descriptive icons.
4. **Archive:** Completed/abandoned items. No icon changes needed.

### Icon Guidelines
- 💰 Money/Finance related
- 💼 Career/Jobs
- 📈 Trading/Investing
- 💪 Health/Fitness
- 🎸 Music/Hobbies
- 🔮 Ideas/Planning
- 🗝️ Life goals/Bucket list
- 🎟️ Work/Professional
- 📘 Learning/Books
- 🌟 Food/Recipes
