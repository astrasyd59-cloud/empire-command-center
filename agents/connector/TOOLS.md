# TOOLS.md — Connector

## Available Tools

### Notion Database (Relationship Tracker)
**Status:** ✅ Active  
**Purpose:** Central relationship and opportunity database  
**Data Tracked:**
- Contact profiles (names, details, context)
- Relationship stages and history
- Interaction logs and notes
- Opportunity pipeline
- Event calendar integration

**Access:** Via Notion API

---

- Monitor new event announcements
- RSVP management

---

### Calendar Integration
**Status:** ✅ Active  
**Purpose:** Sync social commitments and track availability  
**Capabilities:**
- Read/write social events
- Block social time
- Send reminders for upcoming commitments
- Avoid double-booking
