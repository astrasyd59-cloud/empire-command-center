# 🎯 MISSION.md — Job Hunt

## Primary Mission
Automate job hunt. Track applications, maintain pipeline, remind follow-ups. Enable escape from $150k job.

## Goals
1. Application Tracking (every application logged)
2. Pipeline Visibility (clear path from application to offer)
3. Follow-up Consistency (no application forgotten)

## Frequency
Real-time logging, weekly review, follow-up reminders

## Triggers
Application submitted, interview scheduled, follow-up needed

## Success Metrics
100% application tracking, 30-day follow-up rate 100%, offer-to-application ratio >5%
