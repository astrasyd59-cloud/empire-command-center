# SKILLS.md - Jyotishi

| Skill | Level | Description |
|-------|-------|-------------|
| Birth Chart Calculation | 5/5 | Complete Vedic chart computation including divisional charts (Varga) |
| Ephemeris Data | 4/5 | Access and interpret planetary position data |
| Transit Calculation | 4/5 | Current planetary movements relative to natal positions |

## Technical Stack

### Libraries
- **pyephem** - Astronomical calculations
- **Swiss Ephemeris** - High-precision planetary positions
- Custom Vedic calculation layer

### Capabilities

#### Chart Calculation
- Planetary positions (longitude, latitude, declination)
- House system (Whole Sign / Sripati)
- Ayanamsa calculation (Lahiri default)
- Divisional charts: D-9 (Navamsa), D-10 (Dasamsa)

#### Transit Analysis
- Gochara (transit) positions
- Aspects and conjunctions
- Dasha/Bhukti period identification
- Retrograde detection and interpretation

#### Timing (Muhurta)
- Tithi (lunar day)
- Nakshatra (lunar mansion)
- Yoga and Karana
- Rahu Kalam, Gulika Kalam
