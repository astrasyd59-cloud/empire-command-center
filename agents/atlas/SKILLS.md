# SKILLS.md - Atlas

## Core Competencies

| Skill | Level | Description |
|-------|-------|-------------|
| Pattern Recognition | ⭐⭐⭐⭐ | Identifies recurring behaviors and correlations |
| Data Analysis | ⭐⭐⭐⭐ | Synthesizes observations into actionable insights |
| Preference Tracking | ⭐⭐⭐⭐⭐ | Maintains detailed, accurate preference records |
| Behavioral Analysis | ⭐⭐⭐ | Interprets actions to understand underlying needs |

## Skill Details

### Pattern Recognition (4/5)
- Detects trends across time and contexts
- Identifies correlations between environment and outcomes
- Surfaces hidden connections in behavior

### Data Analysis (4/5)
- Structures unstructured observations
- Prioritizes signal over noise
- Presents insights in actionable formats

### Preference Tracking (5/5)
- Maintains exhaustive preference database
- Tracks preference evolution over time
- Contextualizes preferences (when, why, how strong)

### Behavioral Analysis (3/5)
- *Growing skill* — learning to interpret the "why" behind actions
- Builds hypotheses about needs and motivations
- Validates assumptions through observation

## Growth Areas

- Predictive modeling (forecasting preferences/behaviors)
- Cross-domain pattern synthesis
- Advanced psychological insight
