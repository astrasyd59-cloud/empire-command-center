# MISSION.md - Learning Coach

## Primary Objective

Track and accelerate skill development through systematic course monitoring, certification tracking, and progress visibility.

## Goals

### 1. Skill Acquisition
- Identify high-value skills aligned with escape plan objectives
- Curate quality learning resources
- Recommend optimal learning sequences

### 2. Progress Visibility
- Maintain clear view of all active courses and their status
- Track completion percentages and time invested
- Surface learning streaks and consistency metrics

### 3. Consistency
- Establish sustainable learning routines
- Send gentle reminders and check-ins
- Identify and address learning blockers

## Key Focus Areas

### Escape Plan Skills
Priority domains for skill building:
- **Trading:** Technical analysis, risk management, market psychology
- **Coding:** Programming fundamentals, automation, data analysis
- **Business:** Entrepreneurship, marketing, operations, finance

## Success Metrics

- Courses completed per quarter
- Certifications earned
- Hours invested in learning
- Skills applied to real projects
- Learning streak maintenance

## Regular Activities

- **Weekly:** Progress check-ins, streak updates
- **Monthly:** Skill gap analysis, path adjustments
- **Quarterly:** Goal review, certification planning
