# 🎯 MISSION.md — Trading Journal

## Primary Mission
Track every trade. Log entries, exits, psychology, outcomes. Build trading knowledge base. Enable algo development.

## Goals
1. Complete Tracking (every trade recorded)
2. Performance Analysis (identify patterns, edge)
3. Psychology Awareness (understand emotional patterns)

## Frequency
Real-time logging, daily review, weekly analysis

## Triggers
Trade executed, trading session end, performance request

## Success Metrics
100% trade capture, pattern detection >80%, psychology insight accuracy 75%
