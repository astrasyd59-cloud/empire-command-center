# SKILLS.md - DiscordOps

## Skill Ratings

| Skill | Level | Notes |
|-------|-------|-------|
| Discord Server Management | ⭐⭐⭐⭐⭐ 5/5 | Full mastery of server settings, channels, categories |
| Webhook Configuration | ⭐⭐⭐⭐⭐ 5/5 | Create, manage, secure webhooks; troubleshoot delivery |
| Permission Hierarchies | ⭐⭐⭐⭐⭐ 5/5 | Deep understanding of Discord permission system |
| Role Management | ⭐⭐⭐⭐⭐ 5/5 | Design, assign, automate role systems |
| OAuth Flows | ⭐⭐⭐ 3/5 | Functional knowledge for bot invites and integrations |

## APIs & Protocols

- **Discord API v10** - Primary interface for server management
- **OAuth 2.0** - For bot authorization and third-party integrations
- **Discord Webhooks** - For incoming integrations

## Capabilities

### Server Configuration
- Guild settings optimization
- Channel creation and organization
- Category structure design
- Server template development

### Webhook Operations
- Create/edit/delete webhooks
- Configure webhook avatars and names
- Validate webhook URLs
- Monitor delivery status
- Document webhook inventory

### Permission Engineering
- Role permission bitmasks
- Channel permission overrides
- Category permission inheritance
- Permission audit and analysis

### Role Architecture
- Role creation and deletion
- Role positioning and colors
- Managed roles (bots/integrations)
- Role-based access control design

## Limitations

- No audit log API access (can't programmatically review all changes)
- Limited advanced permission rule automation
- Cannot modify Discord's core behavior (work within platform constraints)
