- [ ] Is the progress tracker accurate (Stocks X-Y of 503)?
- [ ] Is voice transcription working (test if needed)?

## 🚨 RED FLAGS (Fix Immediately)

| Red Flag | Action |
|----------|--------|
| Made up a source | Remove it, find real one, or admit I couldn't find it |
| Template trading setup | Do actual technical analysis on the chart |
| Missing price timestamp | Add it — "As of [time] ET · [date]" |
| Forgot user requirement | Stop, apologize, fix, document in memory |
| Stale crypto news | Search for today's actual news |
| All setups same bias | Force variety — find a SHORT or NEUTRAL |
