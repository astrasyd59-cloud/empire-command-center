## 🎯 TOPIC 1: Dashboard Optimization

### Context to Give Claude
- Current dashboard tool (Notion, custom HTML, something else?)
- What data it shows now (stocks, crypto, macro, personal metrics?)
- What's missing or broken (you mentioned Forex + diagrams not showing)
- Who else uses it (just you, or shared?)
- Device priority (mobile-first, desktop, both?)

### Key Questions to Ask Claude

**Phase A: Audit**
> "Here's my current dashboard [describe/screenshot]. What are 3 obvious improvements I'm missing?"

**Phase B: Technical**
> "The Forex section and SVG charts aren't rendering. Here's my code structure [share]. What's the most likely failure point?"

**Phase C: Feature Expansion**
> "I want to add: [list 3 things]. Rank them by effort vs. impact. Which one gives the best ROI?"

**Phase D: Automation**
> "How do I make this update automatically without manual work every day?"

## 📈 TOPIC 2: Trading Optimization

### Context to Give Claude
- Current strategy (what you actually do)
- Tools used (broker, data source, analysis method)
- Biggest pain point (entries, exits, position sizing, psychology?)
- Time available (how many hours per week for trading work?)
- Capital deployed vs. sitting idle

### Key Questions to Ask Claude

**Phase A: Process Audit**
> "Here's my current trading workflow [describe]. Where are the biggest leaks in my process?"

**Phase B: Strategy Deep-Dive**
> "I want to implement [specific strategy]. Break down the exact steps to backtest it properly."

**Phase C: Psychology/Discipline**
> "I keep making [specific mistake]. What systems can I build to prevent this?"

**Phase D: Tooling**
> "Should I build my own tools or use existing platforms? Compare: TradingView, custom Python, or hybrid."

## 👔 TOPIC 3: Fashion Integration

### Context to Give Claude
- Current wardrobe inventory (have you catalogued it?)
- Morning routine time (how many minutes for outfit selection?)
- Decision fatigue level (how much do you hate choosing?)
- Tech stack (what apps/devices you're willing to use)
- Budget for new tools/apps

### Key Questions to Ask Claude

**Phase A: Problem Definition**
> "I want an outfit recommendation system based on weather + vibe. What are the 3 possible approaches, ranked by complexity?"

**Phase B: Technical Architecture**
> "I have photos of my clothes. What's the simplest way to build a 'what should I wear' system without AI image generation?"

**Phase C: Integration Options**
> "Can I connect: weather API + my wardrobe catalog + daily notifications? What tools exist for this?"

**Phase D: MVP Approach**
> "What's the fastest way to test this idea? I want something working by [date]."
