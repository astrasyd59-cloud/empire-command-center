# Price Feed Integration Options for Astra

## Option 1: Yahoo Finance (Free, Delayed 15min)
- Use: `yfinance` Python library
- Pros: Free, reliable, covers FX/stocks/crypto
- Cons: 15-min delay, not real-time

## Option 2: Alpha Vantage (Free Tier)
- API Key required
- 25 calls/day free
- Real-time FX data available

## Option 3: TradingView Integration
- You have access
- Best charts
- Can export data via webhook/pine script

## Option 4: Broker APIs
- City Index API
- CMC Markets API
- Direct from your trading accounts

## Recommendation
Set up Yahoo Finance (free) for daily reports, use TradingView screenshots for real-time analysis (like you just did).
